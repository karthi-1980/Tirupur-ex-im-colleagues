# Tirupur Ex-Im Colleagues Website

A professional website for sharing expertise in Indian imports, exports, customs matters, and foreign trade policies.

## Features
- Professional homepage with navigation
- Contact information for Karthikeyan Mayavan
- Responsive design for all devices
- Smooth scrolling and animations

## How to Start
1. Double-click `launch.bat` to open the website
2. OR double-click `index.html` directly
3. The website will open in your default browser

## How to Make Website Live Online

### GitHub Pages (FREE & RECOMMENDED)
1. Create GitHub account at https://github.com
2. Create repository named "tirupur-ex-im-colleagues"
3. Upload all files to the repository
4. Enable GitHub Pages in repository settings
5. Your website will be live at:
   https://[your-username].github.io/tirupur-ex-im-colleagues
